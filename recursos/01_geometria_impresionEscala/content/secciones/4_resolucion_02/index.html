Practica con la otra imagen

A continuación se muestra la otra foto que se presentó en la primera página de esta unidad. Recuerda que esta imagen mide 400x600 px.

Ahora imagina que tuvieras que editar esta foto con algún software de edición de imágenes. Típicamente, este tipo de aplicaciones ofrecen varios modos de selección de una área determinada de la imagen, para poder copiar y pegar dicha área en otra ventana de edición.

  • Selección normal: Se puede arrastrar el cursor para seleccionar de manera libre cierta área de la imagen.
  • Selección con razón fija: Se arrastra el cursor para seleccionar una área de la imagen que presenta una razón predeterminada entre sus lados.
  • Selección por tamaño fijo: Se indica el valor que deben tener ambos lados y solo se puede mover el área indicada, sin cambiar su tamaño.

A continuación se ofrece un espacio interactivo que simula este tipo de herramientas de edición. Explora las distintas maneras de selección y decide cómo recortarías la foto a fin de optimizar su impresión (dando a sus lados la razón requerida pero sin recortar a la persona retratada).

Primero, elige uno de los tres modos de selección. Haz los cálculos necesarios para determinar cuál deben ser el ancho y alto de tu imagen. Dependiendo del modo que escogiste, podrás cambiar el tamaño del área seleccionada arrastrando el círculo gris obscuro. También podrás cambiar la ubicación del área seleccionada (es decir, sus coordenadas \(x\) y \(y\)) arrastrándola con el ratón. Cuando estés satisfecho con el área de recorte, pulsa 'Recortar'.
